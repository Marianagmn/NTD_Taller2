Proyecto Web Estático
Este proyecto corresponde al Taller 2 de la asignatura NTD y consiste en el desarrollo de un sitio web estático utilizando HTML, CSS y JavaScript.
El sitio presenta una temática inspirada en el mundo del anime, incluyendo una página principal, fichas informativas y una sección dedicada a un personaje específico "Zoro de one piece".
1. Estructura del Proyecto

│
├── Inicio.html
├── ficha.html
├── zoro.html
├── styles.css
├── script.js

2. Descripción de Archivos
2.1 Inicio.html
Página principal del sitio web. Contiene la estructura base de navegación y presentación general del contenido.
2.2 ficha.html
Página donde se muestra información detallada tipo ficha técnica o descriptiva.
2.3 zoro.html
Página dedicada específicamente a un personaje, incluyendo contenido visual e informativo.
2.4 styles.css
Archivo de estilos encargado del diseño visual del sitio: colores, tipografías, distribución y diseño responsive (si aplica).
2.5 script.js
Archivo JavaScript que maneja las interacciones dinámicas, eventos y funcionalidades adicionales del sitio.
2.6 Carpeta assets
Contiene todos los recursos multimedia utilizados en el proyecto, como imágenes, logos y elementos gráficos.
3. Tecnologías Utilizadas
* HTML5 – Estructura del sitio.
* CSS3 – Estilos y diseño visual.
* JavaScript – Funcionalidad dinámica e interacción.
4. Objetivo del Proyecto
Aplicar los conceptos básicos de desarrollo web, implementando estructura semántica en HTML, diseño visual con CSS e interactividad mediante JavaScript, organizando correctamente los recursos del proyecto.
5. Cómo Ejecutar el Proyecto
1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo 'Inicio.html' en un navegador web.
No se requiere instalación adicional ni servidor.
